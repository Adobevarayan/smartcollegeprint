# Smart College Print Management System

A web-based system to manage student print requests in colleges.

## Features
- Student login & registration
- Printer selection
- Print job submission
- Cost calculation
- Job history tracking

## Tech Stack
- HTML, CSS, JavaScript
- PHP (PDO)
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Place project in `htdocs`
3. Import `database/smart_print.sql` in phpMyAdmin
4. Start Apache & MySQL
5. Open `http://localhost/smart-print/public/`
